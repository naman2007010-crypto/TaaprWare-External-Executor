# Required Tools for Reverse Engineering

The automated download failed (likely due to network restrictions or changing URLs). Please download these two tools manually:

## 1. x64dbg (Debugger)
- **Website**: https://x64dbg.com/
- **Direct Download**: [Click Here](https://sourceforge.net/projects/x64dbg/files/latest/download)
- **Action**: Download the zip, extract it to this `tools` folder.

## 2. Cheat Engine (Memory Scanner)
- **Website**: https://cheatengine.org/
- **Direct Download**: [Click Here](https://github.com/cheat-engine/cheat-engine/releases)
- **Action**: Download the Installer (`CheatEngineXX.exe`) and install it.
- **Note**: You may need to disable Real-time Protection temporarily as it's often flagged as a "HackTool".

## Where to put them?
- Extract x64dbg to: `tools/x64dbg/`
- Install Cheat Engine anywhere (e.g., standard Program Files)

Once installed, follow the `reverse_engineering_guide.md` in the parent folder!
